# Security

## Summary

This showcase uses demonstration values and placeholder infrastructure names.

Production work should use temporary limited AWS access. Root credentials and AdministratorAccess are not required.

## Access Principles

- Use temporary IAM user, IAM role, IAM Identity Center, or federated access.
- Keep permissions limited to the approved monitoring scope.
- Approve production changes before apply.
- Remove temporary access after delivery.

## Public Showcase Safety

This public material is written for buyers and keeps production delivery details separate from the showcase.
